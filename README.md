# About
**Stremio Stinger Pro** is a metadata addon for the Stremio and other supported media libraries. It automates the detection of mid-credits and post-credits scenes (*stingers*) for feature films. It provides users with immediate, actionable advice on whether they should stay seated during the credits or if it is safe to stop playback.
```
https://stremio-addons.net/addons/stremio-stinger-pro
```

## Sources
In order of calling
1. AfterCredits.com
2. MediaStinger.com
3. TMDB (fallback with personal API configuration)

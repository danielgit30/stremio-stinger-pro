# 2024-05-24 - Sequel Setup Checkbox Feature

**Learning:** Adding new functional options to existing user interfaces requires careful integration to ensure accessibility and maintaining the existing visual language. Using descriptive small text annotations next to checkboxes provides clear context to the user (e.g. `Show Sequel Setup <small>(Works correctly only when Source is AfterCredits)</small>`).
**Action:** When expanding UI configurations, always ensure the new components gracefully adapt to dynamic updates (live previews) and follow established styling patterns like `checkbox-group`.
